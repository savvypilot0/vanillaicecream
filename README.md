# ответы на вопросы, которые могут у вас появится.
### как скачать?
чтобы скачать готовый инстанс, идите на вкладку [релизов](https://github.com/savvypilot0/vanillaicecream/releases).<br>
для нормальной работы нужно большое количество оперативной памяти (по умолчанию выделено минимум 6 гб).
### работает на линукс?

### сборка крашится, что делать?
существует много причин такий приколов, чаще всего это происходит из-за несовместимой версии джавы.<br>
если у вас возникают проблемы, открывайте [issue](https://github.com/savvypilot0/vanillaicecream/issues).
